# FinFET-Based SRAM Design (22nm)

This project presents the design and analysis of a 6T SRAM cell using FinFET technology at 22nm.

## Tools Used
- Cadence Virtuoso
- Spectre Simulator

## Key Features
- Comparison across 90nm, 45nm, 22nm
- SNM analysis
- Power and delay evaluation
